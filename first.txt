Gitの練習をしていきます。

Gitを使用した一連のワークフロー
1. ファイルの変更をステージングエリアに追加する
2. ローカルリポジトリにこみっとする
3. リモートリポジトリにプッシュする

ステージングエリア
ディレクトリの中のどのファイルをcommit, pushするかを
選択する

commit
ローカルリポジトリに登録する
version管理される

push
リモートリポジトリに変更を保存する
同じプロジェクトに参加している他のユーザに共有できる

git remote add
ローカルリポジトリをリモートリポジトリに登録できる。

.gitignore
version管理したくないファイルを指定できる
例えば自動生成ファイルやパスワードが記載されているファイル
をアップロードしないようにできる。
